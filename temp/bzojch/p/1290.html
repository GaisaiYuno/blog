[Ctsc2009]序列变换

时间限制：30s      空间限制：162MB

题目描述

输入格式

输出格式

样例输入

3 6 2 2
1 4 6

样例输出

1

提示

可以将序列变换为2 4 6或者1 3 5。前者变换代价为1，后者为2。因此最小TransformCost为1。 对于10%的数据 , N ≤ 100, Q ≤ 10000, 1≤ A, B ≤ 100。 对于30%的数据 , N ≤ 10000, Q ≤ 10000, 1≤ A, B ≤ 100。 对于60%的数据 , N ≤ 10000, Q ≤ 109, 1≤ A, B ≤ Q。 对于100%的数据, N ≤ 500000, Q ≤ 109, 1≤ A, B ≤ Q。
